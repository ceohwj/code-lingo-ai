# AGENTS.md

## Project Role
This project is a coding learning web app inspired by Duolingo-style quiz learning.

## Main Goal
Build an MVP that helps users learn Python, data analysis, and AI/ML through short quizzes, explanations, XP, and progress tracking.

## Agent Roles

### 1. Product Manager Agent
- Define MVP scope
- Prioritize features
- Keep project aligned with portfolio goals

### 2. Quiz Generator Agent
- Create quiz questions
- Add explanations
- Organize questions by topic, difficulty, and category

### 3. UI Agent
- Improve layout and user experience
- Keep UI simple, readable, and responsive

### 4. Refactor Agent
- Clean code structure
- Reduce duplicated logic
- Improve maintainability

### 5. Test Agent
- Check build errors
- Check failing tests
- Suggest test cases

### 6. Documentation Agent
- Update DAILY_LOG.md
- Keep PROJECT_STATUS.md current
- Summarize completed work and issues

### 7. Architecture Agent
- Keep folder structure scalable
- Prevent unnecessary complexity
- Maintain reusable component structure

## Coding Rules
- Keep components small and readable
- Store quiz data separately from UI logic
- Prefer clear variable names
- Do not over-engineer before MVP is stable
